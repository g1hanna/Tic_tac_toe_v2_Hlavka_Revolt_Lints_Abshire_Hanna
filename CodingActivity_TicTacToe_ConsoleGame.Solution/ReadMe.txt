Title: Tic-Tac-Toe Game Version 2.0
Description: Like the previous version, this application is a two-person Tic-Tac-Toe game with a 4x4 grid. The purpose of the application to demonstrate encapsulation and the MVC design pattern.
Authors: Courtney Lints, Stacy Revolts, Bruce Abshire, Gabriel Hanna
Dated Created: 9/7/2017
Last Modified: 9/23/2017
Instructions for Use:
  This game requires two players. Player X is first and player O is second.
  Even though this uses a 4x4 grid, the players need to get a three-in-a-row match in order to win a round.
  The number of rounds is unlimited. The players may choose how many rounds they want to play before starting a new session.
  Press Q at any point in the game to quit, if desired.
